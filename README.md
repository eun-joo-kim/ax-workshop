# ax-workshop

김은주의 AX 워크숍 · 포트폴리오 · 발표자료 모음.

## 주요 페이지

| URL | 설명 |
|-----|------|
| [ppt01](https://eun-joo-kim.github.io/ax-workshop/ppt01/) | 포트폴리오 — 프레젠테이션 자동화 에이전트 (청중 화면) |
| [ppt02](https://eun-joo-kim.github.io/ax-workshop/ppt02/) | 포트폴리오 발표자 뷰 (PPT 01 과 쌍, BroadcastChannel 동기화) |
| [ppt03](https://eun-joo-kim.github.io/ax-workshop/ppt03/) | AX Workshop Week 1 |

## 사용법

**ppt01 + ppt02 쌍**: 같은 브라우저 두 탭에서 열면 `BroadcastChannel`로 자동 동기화. 빔프로젝터에 ppt01, 노트북에 ppt02를 띄우면 됩니다.

**키보드 단축키**
- `→` / `Space` / `Enter` — 다음 슬라이드
- `←` — 이전 슬라이드
- `↓` / `↑` — B-슬라이드 드릴다운 (ppt01 전용)
- `T` — 텔레프롬프터 모드 (ppt02)
- `F` — 풀스크린 / 스크립트 전체화면

## 기술 스택

HTML5 + CSS3 + Vanilla JS. 외부 의존성 0 — Pretendard · 온글잎 영현 폰트 base64로 HTML 내장 (오프라인 동작).
