Gemini CLI 설치 — Mac

🍎 Mac — 터미널 사용
1
Homebrew 확인 · 설치
버전 확인: brew --version → 숫자 나오면 OK
2
Node.js 확인 · 설치
node --version / npm --version
없으면 → brew install node
3
Gemini CLI 설치
npm install -g @google/gemini-cli
4
설치 확인
gemini --version → 버전 나오면 완료 ✅
Homebrew 설치 명령어
# Homebrew 설치 (없을 경우) /bin/bash -c "$(curl -fsSL \ https://raw.githubusercontent.com/\ Homebrew/install/HEAD/install.sh)" # Node.js 설치 brew install node # Gemini CLI 설치 npm install -g @google/gemini-cli # 설치 확인 gemini --version
07

Gemini CLI 설치 — Windows

🪟 Windows — PowerShell 사용
1
Node.js 설치
nodejs.org → LTS 버전 다운로드 후 설치
2
설치 확인
PowerShell에서 node --version / npm --version
3
Gemini CLI 설치
npm install -g @google/gemini-cli
4
설치 확인
gemini --version → 버전 나오면 완료 ✅
PowerShell 명령어
# Node.js 버전 확인 node --version npm --version # Gemini CLI 설치 npm install -g @google/gemini-cli # 설치 확인 gemini --version
💡
PowerShell은 관리자 권한으로 실행하세요 (우클릭 → "관리자 권한으로 실행")
