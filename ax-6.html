Claude Code란?
🤖
Anthropic이 만든 AI 코딩 도구 — ChatGPT처럼 대화하면서 실제 파일을 생성·수정·실행까지 해주는 것이 핵심 차이
구분ChatGPT / 웹 ClaudeClaude Code
동작 방식 브라우저에서 대화 내 컴퓨터에서 직접 파일 생성·수정
결과물 텍스트 답변 (복사해서 붙여넣기) 실제 파일이 내 폴더에 저장됨
활용 범위 질문·요약·번역 위주 문서, PPT, 웹사이트, 자동화까지
비용 무료/유료 플랜 Anthropic API 사용량 기반 (Pro $20/월)

실무 꿀팁
  • 대화를 이어가라 — 한 번에 완성하려 하지 말고, 여러 번 수정 요청하며 완성도 높이기
  • 구체적으로 요청하라 — "보고서 만들어줘" → "A4 2장, 표 포함, 파란색 톤의 마케팅 보고서 만들어줘"
  • 결과물을 바로 확인하라 — HTML은 브라우저에서, DOCX는 Word에서 바로 열어서 확인
  • 컨텍스트를 활용하라 — 이전 대화 내용을 기반으로 후속 작업 요청 가능
--dangerously-skip-permissions 옵션
claude --dangerously-skip-permissions
  • 파일 생성, 수정, 명령어 실행 등을 매번 승인하지 않아도 됨
  • 실습할 때 흐름이 끊기지 않아서 편리

Q. 코딩을 전혀 모르는데 따라할 수 있나요?
네. Claude Code는 코딩을 대신 해주는 도구입니다. "~만들어줘"라고 말하면 Claude가 코드를 작성합니다. 이 수업에서 코드를 직접 짤 일은 없습니다.
Q. Claude Code는 무료인가요?
Claude Code 자체는 무료로 설치할 수 있지만, 사용할 때 Anthropic API 비용이 발생합니다. Pro 플랜($20/월)에 가입하면 실습 수준에서는 충분합니다.
Q. 설치가 안 돼요 / 에러가 나요.
가장 흔한 원인은 Node.js가 설치되지 않은 경우입니다. node -v를 터미널에 입력해서 버전이 나오는지 확인하세요.
Q. "Permission denied" 에러가 나요.
Mac에서 npm 설치 시 권한 문제가 생길 수 있습니다. 명령어 앞에 sudo를 붙여보세요: sudo npm install -g @anthropic-ai/claude-code
FAQ · 사용법

Q. Claude Code가 멈춘 것 같아요. 반응이 없어요.
Esc를 눌러서 중단하고, /clear로 대화를 초기화한 후 다시 시도하세요. 긴 작업 중에는 1~2분 기다려보세요.
Q. 결과물이 마음에 안 들면 어떻게 하나요?
그냥 자연어로 수정 요청하면 됩니다. "표를 추가해줘", "색상을 파란색으로 바꿔줘", "2페이지를 다시 만들어줘"처럼 원하는 부분을 구체적으로 말하세요.
Q. "Allow?" 팝업이 계속 뜨는데 안 뜨게 할 수 있나요?
claude --dangerously-skip-permissions로 실행하면 모든 권한을 자동 허용합니다.
Q. 만든 파일을 다른 사람한테 보내려면요?
Claude Code가 생성한 파일은 내 컴퓨터 폴더에 실제 파일로 저장됩니다. DOCX, HWP, PPTX 등 일반 파일이므로 이메일, 메신저로 바로 전송할 수 있습니다.
Q. 대화 내용이 날아갔어요.
claude --continue를 입력하면 직전 대화를 이어서 할 수 있습니다. 이전에 생성한 파일은 폴더에 그대로 남아 있으므로 새 대화에서도 작업을 이어갈 수 있습니다.
Q. ChatGPT처럼 이미지도 보여줄 수 있나요?
Claude Code는 이미지를 화면에 보여주지는 않지만, 이미지를 읽고 분석할 수 있고, HTML/PPT 안에 이미지를 넣을 수 있습니다. Cmd+V로 스크린샷을 붙여넣어서 "이 디자인 참고해서 만들어줘"도 가능합니다.
